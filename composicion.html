La base compositiva

5x6, esa es la retícula base que forma nuestro isotipo. Y es la que ha dado vida a nuestro sistema de composiciones y retículas. Teniendo en cuenta los ajuste ópticos necesarios siempre se ha seguido esta retícula base para generar ritmo y bloques compositivos.

Desde un principio conocemos la amplitud de la marca y las necesidades reticulares que esta tendría. Necesitamos generar un sistema compositivo que funcione y que haga reconocer la marca desde la web a un poster por la calle. Y para ello hemos comenzado con el isotipo. La primera marca y la más importante de Madrid Open City fue su isotipo y es en base a su estructura que gira toda la marca. El desarrollo del mismo se explica en su parte correspondiente en el manual. Una vez conseguido una idea del isotipo desarrollada hemos trabajado el mismo con una retícula.

Poco a poco hemos ido simplificando la estructura a formas básicas hasta llegar al cuadrado como módulo estructural. Es este cuadrado el que marca la generación de la diagonal con respecto a la altura y la relación entre los tres rectángulos que forman el isotipo. Una vez generada esta estructura de 3x4 para cada rectángulo y 5x6 somos capaces de generar la estructura de cualquier producto gráfico o audiovisual. Con esta estructura generamos ritmo vertical, horizontal y diagonal, ya que los rectángulos siguen siempre un ángulo de 63º.

Relaciones

Cogiendo esta composición básica es cuando empezamos a generar estructuras complejas. Adaptando el tamaño del módulo según sea necesario generamos ritmos horizontales que se relacionan en un ⅚ con los ritmos verticales.

Las cajas de texto tendrán los tamaños necesarios aunque siguiendo la retícula que se genere para el formato específico. Siempre se intentarán generar múltiplos del 5x6 o 3x4 tanto verical como horizontal.

La relación entre texto, imágenes y recursos gráficos se hará siempre siguiendo el mismo módulo cuadrado con el tamaño generado para cada formato. Así para un A4 se pueden llegar a usar dos módulos de distancia y para la web uno.

Retícula

4+1

Representa el uso en las columnas para la web. Usando la estructura base generamos una cuadrícula de 12 columnas para la web. 4 es la columna, 1 es la calle y 6 y sus múltipos son las divisiones horizontales (usadas normalmente para encabezados).

4x4

Para una retícula en papel se adapta una vez más el módulo y en base a su tamaño se generan márgenes, columnas y filas. El sistema permite adaptar la retícula a cualquier formato. En este caso la diagonal se puede usar como elemento de diseño terciario, sin afectar a la legibilidad y obviamente sin generar cajas de texto diagonales.

Retícula para un DIN A4

Animado

Para las animaciones se emplea el mismo sistema modular y aparecen con mayor importancia las diagonales. Es en base a estas que se mueve el isotipo y que aparecen los textos informativos en los identificativos. Con esta misma retícula y fórmula se segmenta el espacio y ajusta para que se puedan generar identificativos desde banners en spotify hasta anuncios de YouTube.
