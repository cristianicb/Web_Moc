Retícula corporativa

El isotipo como modular

Tanto en web como impreso se usará el isotipo de la marca como módulo base. El isotipo está compuesto sobre una rejilla cuadrada de 5 horizontales por 6 verticales. Va a ser esta rejilla la que nos permita utilizar el isotipo para crear retículas. El submódulo cuadrado se usará para marcar el intercolumnado, los márgenes y el espacio horizontal entre componentes para la web.

Digital

Desktop First

Se ha generado una retícula de doce columnas, que permite gran versatilidad de anchos y permite generar el responsive facilmente. Se trabajará por bloques y fracciones. ⅓, ¼ o ½ serán los tamaños más usados. Cada bloque tendrá de forma sencilla su reposicionamiento a la versión móvil.

Papelería

Tomamos como punto de inicio la retícula de un documento A4, sin duda alguna el formato más usado para la papelería y para impresión de documentos. Repitiendo el isotipo ocho veces de forma horizontal y 9 de forma vertical conseguimos una retícula de 40 por 56 cuadrados de la rejilla inferior (la usada para formar el isotipo). Lo que conseguimos de este modo es generar uno margen de 2 unidades y columnas y filas de 4 unidades. Sobre esta base se construye una rejilla de columnas con calles iguales a la interlínea usado para el texto corrido.

Para documentos con tamaños especiales se vuelve a usar el mismo sistema, repitiendo el isotipo de forma horizontal y usando un mínimo de 2 unidades de la rejilla base para los márgenes. Si el formato genera una incoherencia entre la vertical y la horizontal (queda un numero con decimales para las unidades verticales) se ajusta la caja a la parte superior.
